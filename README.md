# AI-Augmented Learning Prompts

This repository contains ready-to-use prompts to run AI-augmented, inverted-pedagogy-style learning sessions.

## Why

- It's **Faster** because you only study what you need to know.
- It's **Just-in-time**. There is so much information out there that we cannot learn everything just in case. This approach allows you to defer learning until you really need it, focusing on the most relevant content, across various sources.
- It's **More fun and engaging** because it uses active learning techniques and Socratic questioning.
- It's **personalized** because it adapts to your learning style, personality, tastes, background, and needs.
- It's **More effective** because it's active learning and connects sources, helping you make neural connections.

## What this means

- **AI-augmented**: learners use an LLM through structured prompts inspired by the [Accelerate Your Learning with ChatGPT MOOC](https://www.coursera.org/learn/learning-chatgpt).
- **[Training from the Back of the Room](https://www.trainingfromthebackoftheroom.com/)**: learning plan follows the 4Cs model (Connections, Concepts, Concrete Practice, Conclusions) and uses active learning techniques.
- **Socratic Questioning**: LLM acts as a Socratic teacher, asking questions to assess and deepen understanding.
- **[Perfection Game Feedback](https://thecoreprotocols.org/protocols/perfectiongame)**: LLM provides feedback using the Perfection Game protocol, to give learners a clear self-assessment.

Optional learning supports:

- Note taking
- Flashcards

## How to use

![Flow diagram representing the learning experience](flow.svg)

1. Define a rough learning goal and add enough personal context for the AI to adapt the learning experience.
2. Build a knowledge base using one of two paths:
	- discover sources if you do not yet know what to study,
	- or upload your own selected sources if you already have them.
3. Check context-window fit, then prioritize or split by sub-topics when needed.
4. Optionally run an 80/20 exploration pass (top insights and surprises) and refine your learning goal.
5. Run a 10-question self-assessment with Socratic questioning and Perfection Game feedback.
6. Use assessment results to build a 4Cs-based Socratic learning path and run session by session.
7. Continue with extension sessions, deliberate practice, or deeper open discussion.
8. Optionally add note taking and flashcards throughout the journey.

## Typical time expectation

- Building the knowledge base: 0 to 2 hours
- Main takeaways and surprises: about 10 minutes
- Self-assessment: about 1 hour
- Socratic learning: 1 to 2 hours
- **Total:** between 15 minutes to 5 hours, depending on how much content you have and how deep you want to go.
